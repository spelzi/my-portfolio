# Contributor Covenant Code of Conduct

## Our Pledge

We, as members, contributors, and maintainers, pledge to make participation in this project a harassment-free experience for everyone, regardless of:

- Age
- Body size
- Disability
- Ethnicity
- Gender identity and expression
- Level of experience
- Nationality
- Personal appearance
- Race
- Religion
- Sexual identity and orientation

We are committed to acting professionally, respectfully, and collaboratively.

---

# Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and welcoming
- Accepting constructive criticism
- Focusing on what benefits the community
- Showing empathy toward others
- Using inclusive language
- Helping new contributors

Examples of unacceptable behavior include:

- Harassment
- Personal attacks
- Discrimination
- Trolling
- Publishing private information without permission
- Any behavior that is inappropriate in a professional environment

---

# Our Responsibilities

Project maintainers are responsible for:

- Clarifying acceptable behavior
- Reviewing reported incidents
- Taking appropriate action when necessary
- Maintaining a welcoming environment

Maintainers may edit, remove, or reject comments, commits, issues, pull requests, or other contributions that violate this Code of Conduct.

---

# Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Project documentation
- Community interactions related to this repository

---

# Enforcement

Project maintainers may take any action they deem appropriate, including:

- Warning the contributor
- Removing inappropriate content
- Temporarily restricting participation
- Permanently banning repeat offenders
